# ESP32-StockTicker
This is an open-source stock ticker project that I made with an esp32 using Finnhub.io's API.

Wiring Guide:
RST,GPIO 4 (Reset)
CS,GPIO 22 (Chip Select)
DC,GPIO 16 (Data/Command Select)
SDA (MOSI),GPIO 23(SPI Data Out)
SCL (SCK),GPIO 18(SPI Clock)
GND,GND(Ground)
VCC,3V3(Power)
